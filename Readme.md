# Quotely Backend

A simple Express.js + MongoDB backend for saving and managing user quotes.  
Later versions will include AI features for auto-tagging and recommendations.

---

## Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/<your-username>/quotely-backend.git
   cd quotely-backend

2. **Install dependencies**
  npm install

3. **Run in dev mode**
  npm run dev

4. **Run ESLint check**
  npm test
