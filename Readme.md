# Quotely Backend

A simple Express.js + MongoDB backend for saving and managing user quotes.  
Future updates will include AI-powered features like auto-tagging, summarization, and content recommendations.

---

## Setup

1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/quotely-backend.git
   cd quotely-backend

2. Install dependencies
  ```bash
  npm install

3. Start the development server
   ```bash
   npm run dev

4. Run ESLint check
  ```bash
  npm run test
